--SnapBridge Application

To Run Application:
	1. Goto Web.config
	2.In ConnectionStrigs tag, replace Data Source="Server Name" with your Server name for SQL DB
	3.Replace User name and Password in the connection string with the user name and password used to login MMSM

To Run UnitTests:
	1. Goto app.config
	2.In ConnectionStrigs tag, replace Data Source="Server Name" with your Server name for SQL DB
	3.Replace User name and Password in the connection string with the user name and password used to login MMSM
